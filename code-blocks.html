Anonymous Code Blocks

  1. Always use braces and an indented code block with {{if}}, {{while}}, {{for}}

Always use braces and an indented code block with {{if}}, {{while}}, {{for}}

{% highlight scala }
// wrong
if (condition) log.info(“something interesting happened”)
{ endhighlight %}

{% highlight scala }
// wrong
if (condition)
log.info(“something interesting happened”)
{ endhighlight %}

{% highlight scala }
// wrong
if (condition)
log.info(“something interesting happened”)
else
log.info(“nothing happened, really”)
{ endhighlight %}

{% highlight scala }
// right
if (condition) {
log.info(“something interesting happened”)
}
{ endhighlight %}

{% highlight scala }
// right
if (condition) {
log.info(“something interesting happened”)
} else {
log.info(“nothing happened, really”)
}
{ endhighlight %}

The only exception to this rule is when an {{if} is used as substitute for the ternary operator (?) — which is not available in Scala — and when the entire expression fits in a single line,

{% highlight scala }
// right
val level = if (state == “dev”) debugLevel else warnLevel
{ endhighlight %}
